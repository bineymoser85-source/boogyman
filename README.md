# Boogyman

Quick installer for Boogyman.

## Installation

Run the following command to download the installation script:

```bash
wget https://raw.githubusercontent.com/bineymoser85-source/boogyman/main/install.sh
```

Or with curl:

```bash
curl -O https://raw.githubusercontent.com/bineymoser85-source/boogyman/main/install.sh
```

Then make it executable and run it:

```bash
chmod +x install.sh
./install.sh
```

That's it! 🚀
